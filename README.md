Маленький проект, статистическая модель посленулевого общества из книги Семиевие. Расчет вероятности сохранения 7ми независимых рас за 5000 лет после ноля.
Задним умом я понял, что все это бессмысленно, т.к. расы, вероятно, не могут смешиваться физически.
Потом еще для самообразования и т.к. результаты были странные, я замутил юнит тесты. Это мой первый опыт.
